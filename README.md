## Encode Health AI 

Encode Health AI is a full-stack AI-powered health assistant that answers health-related queries using a Large Language Model (LLM).

The application is built with a React (Vite) frontend and a Node.js + Express backend, integrated with the Groq LLaMA model.


##  Features
- AI-based health query answering
- Clean and simple UI
- Fast responses using Groq LLM API
- Full-stack architecture (client + server)


##  Tech Stack
**Frontend**
- React
- Vite
- JavaScript

**Backend**
- Node.js
- Express

**AI Model**
- Groq (LLaMA)
